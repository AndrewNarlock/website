Bible Tract Website Design

The goal of this project was to design a Bible tract website for a client. All images and content were provided by the client. The website was built using the brochure template from my component library (Component Library). It is a fully responsive design and the first image shows the landing page from a desktop PC. The website can be visited here: Bible Tract Website.

The second image shows a contact page as seen on a tablet. The website was designed with mobile users as the main focus and is viewable on any device, as the client uses a QR code to share the website.

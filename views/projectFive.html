Media Deck Prototype

The goal of this project was to create a prototype of a Media Deck. A Media Deck is supposed to be a page that allows users who only have a single monitor to have several, manipulatable, windows that are embedded onto a single page. The prototype can be used here: Media Deck.

The prototype is only capable of embedding YouTube videos as I was more focused on learning about DOM manipulation and JavaScript interactions with this project. A finished product would have the ability to embed any website such as livestreams, articles, etc. The code for this prototype can be seen here: Media Deck Repo

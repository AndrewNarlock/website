Gym Review Website

This project was completed by following The Web Developer Bootcamp 2023 provided by Colt Steele and Udemy.com. It is a full stack web application that allows users to leave reviews for gyms that they have visited. The site is no longer available for users due to hosting price changes, but the following article provides a few screenshots of the website from a users perspective.

I have added a screenshot of the package.JSON file to show what dependencies were used to build the application. It consists of an Express, Node, MongoDB backend and implements several middleware packages for security, routing, and authentication, such as Helmet, Mongoose, and Passport. The front end was built with reusable templates and EJS. Web APIs, such as MapBox, were used for features such as geolocation.

Users are able to authenticate and view from a list of gyms that have been reviewed, as well as creating and editing their own reviews, and commenting on other reviews. Once a user has posted a review for a gym, a heatmap is updated to show other users the location of that gym.

A gym review will display the gyms location, along with the reviewers description and a rating out of ten stars. Other users are able to comment on the review page and add their own rating which is added to the overall gym rating and averaged out.

The owner of a gym review page can change any information or pictures that have been posted with their review. The UI and navigation are extremely user-friendly and responsive. Future improvements to the application could include more customization for user profiles and better searching for specific gym reviews, but since it was built for the sake of learning full stack architecture, the website will probably not be updated any time soon.
